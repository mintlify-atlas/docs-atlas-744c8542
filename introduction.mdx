## How it works

Huashu Design is a `SKILL.md` file — a structured set of instructions and starter components that your AI agent reads before producing design output. It does not require a separate server, UI, or subscription beyond your agent's API usage.

<Steps>
  <Step title="Install the skill">
    Run `npx skills add alchaincyf/huashu-design` in your project directory. The skill is installed into your agent's context.
  </Step>
  <Step title="Give your agent a design task">
    Describe what you want: "Make an iOS prototype for a Pomodoro app with 4 clickable screens" or "Build a product launch animation and export as MP4."
  </Step>
  <Step title="Receive a deliverable">
    The agent reads the skill, follows the Junior Designer workflow (assumptions → placeholders → full execution), and returns an HTML file, MP4, PPTX, or PDF ready for immediate use.
  </Step>
</Steps>

## Agent compatibility

Huashu Design is agent-agnostic. It works with any agent that reads markdown skills:

- **Claude Code** — primary development environment
- **Cursor** — full support via skills protocol
- **Codex** — compatible
- **OpenClaw**, **Hermes**, **Trae** — compatible

<Note>
The skill's internal instructions (`SKILL.md` and `references/*.md`) are written in Chinese. The agent is fully bilingual — all English prompts work correctly. Demos ship in both Chinese and English versions.
</Note>
